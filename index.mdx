AI DevKit is a universal CLI toolkit that bridges the gap between raw AI coding agents and your engineering standards. Install it once, configure any combination of Claude Code, Cursor, Codex, Gemini, and more — and give every agent the same structured workflows, persistent memory, and reusable skills your best developers rely on.

## How it works

<Steps>
  <Step title="Install AI DevKit">
    Run the init wizard. It asks which AI agents you use and which development phases you want — then sets up everything automatically.
    ```bash
    npx ai-devkit@latest init
    ```
  </Step>
  <Step title="Select your agents and phases">
    Choose from Claude Code, Cursor, Codex, Gemini, and more. Pick the development phases that match your workflow (requirements, design, planning, implementation, testing, deployment, monitoring).
  </Step>
  <Step title="Add skills from the registry">
    Extend any agent with community-built skills — debugging guides, TDD workflows, code review checklists, and more.
    ```bash
    npx ai-devkit skill add codeaholicguy/ai-devkit dev-lifecycle
    ```
  </Step>
  <Step title="Start building with AI">
    Your agents now follow structured workflows, remember project knowledge across sessions, and can be monitored and directed from the CLI.
  </Step>
</Steps>
